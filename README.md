# Inception

## Build the images:

```
make build ;
```

## Create and start the containers:

```
make up ;
```

## Display all images, containers, volumes and networks:

```
make ls ;
```

## Build the images, create and start the containers and display the application's stats:

```
make ;
```

## stop the containers:

```
make stop ;
```

## purge Docker on your system:

```
make purge ;
```
